
# 🛡️ Badges de Profil

Ce dépôt comprend une collection de tous les badges disponibles sur GitHub et explique comment obtenir chacun d'eux. Certains sont obtenables, d'autres non, ou encore en phase de test bêta. Ce dépôt classe les badges de `Très Facile` à `Facile`, `Moyen`, et enfin `Difficile`.

Les badges GitHub ont été ajoutés après le lancement du premier hélicoptère martien, [Ingenuity](https://fr.wikipedia.org/wiki/Ingenuity_(hélicopt%C3%A8re)).

> Aujourd'hui, il y a un total de 11 badges sur GitHub, certains ayant également des niveaux bronze, argent et or.
